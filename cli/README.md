# create-project-wind

Create a new Project Wind Tauri app with one command.

## Usage

### npm

```bash
npm create project-wind@latest my-app
cd my-app
npm install
npm run tauri dev
```

### yarn

```bash
yarn create project-wind my-app
cd my-app
yarn install
yarn tauri dev
```

### pnpm

```bash
pnpm create project-wind my-app
cd my-app
pnpm install
pnpm tauri dev
```
